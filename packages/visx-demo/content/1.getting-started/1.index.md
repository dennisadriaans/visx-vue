# visx-vue

visx-vue is a collection of expressive, low-level visualization primitives for Vue 3, ported from the popular [visx](https://airbnb.io/visx) library by Airbnb.

## What is visx-vue?

visx-vue brings the full power of visx's composable visualization primitives to the Vue ecosystem. It provides a set of low-level building blocks that let you compose beautiful charts and data visualizations while retaining full control over your rendering.

### Key Features

- **Vue 3 native** — Built with Composition API and `<script setup>` throughout
- **TypeScript-first** — Complete type safety across all packages
- **Composable** — Build complex charts from small, focused primitives
- **D3-powered** — Leverages the full d3 ecosystem under the hood
- **Familiar API** — 1:1 port of visx's React API to Vue

## Packages

visx-vue is organized as a monorepo with individual packages for each visualization domain:

- `@visx-vue/shape` — SVG shapes (Line, Bar, Area, etc.)
- `@visx-vue/scale` — D3 scale helpers
- `@visx-vue/axis` — Chart axes
- `@visx-vue/grid` — Chart grids
- `@visx-vue/legend` — Chart legends
- `@visx-vue/tooltip` — Tooltips
- `@visx-vue/geo` — Geographic projections
- and many more...
