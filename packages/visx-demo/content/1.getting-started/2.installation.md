# Installation

visx-vue packages are available on npm. Install only the packages you need.

## Prerequisites

- Vue 3.4+
- Node.js 18+

## Install Packages

Install individual packages based on the visualization primitives you need:

```bash
# Core packages
npm install @visx-vue/shape @visx-vue/scale @visx-vue/axis @visx-vue/grid

# Or with pnpm
pnpm add @visx-vue/shape @visx-vue/scale @visx-vue/axis @visx-vue/grid
```

## Quick Example

```vue
<script setup lang="ts">
import { LinePath } from '@visx-vue/shape'
import { scaleLinear } from '@visx-vue/scale'

const data = [
  { x: 0, y: 10 },
  { x: 1, y: 20 },
  { x: 2, y: 15 }
]

const xScale = scaleLinear({ domain: [0, 2], range: [0, 400] })
const yScale = scaleLinear({ domain: [0, 25], range: [200, 0] })
</script>

<template>
  <svg
    width="400"
    height="200"
  >
    <LinePath
      :data="data"
      :x="(d) => xScale(d.x)"
      :y="(d) => yScale(d.y)"
      stroke="steelblue"
      :stroke-width="2"
    />
  </svg>
</template>
```
